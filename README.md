## Running locally

Prerequisite

```sh
pre-commit --version
pre-commit 3.7.0

node --version
v20.9.0

#  run 'corepack enable' to activate corepack
corepack -v
0.20.0
```

Install the dependencies

```sh
make install
```

Start Development Server

```sh
yarn dev
```
